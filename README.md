# christopherkarani/homebrew-orca

Official Homebrew tap for [Orca](https://github.com/christopherkarani/Orca) — a local runtime firewall for AI agents.

## Install

```bash
brew tap christopherkarani/orca
brew install --formula orca
```

## Verify

```bash
orca --version
orca doctor
```

## Upgrade

```bash
brew update
brew upgrade orca
```

## Uninstall

```bash
brew uninstall orca
brew untap christopherkarani/orca
```

## Supported platforms

| Platform    | Architecture |
|-------------|--------------|
| macOS       | arm64, amd64 |
| Linux       | arm64, amd64 |

---

For alternative install methods (script, build from source), see the [main repo](https://github.com/christopherkarani/Orca).
